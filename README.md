# Formulário de Matrícula • Estrelas do Amanhã

Projeto desenvolvido como parte de uma atividade prática do curso **Fullstack da Rocketseat**.

O objetivo desta aplicação é apresentar a interface de um **formulário de matrícula escolar** para o colégio fictício **Estrelas do Amanhã**, com foco em estruturação semântica em HTML e estilização com CSS.

---

## ✨ Sobre o projeto

Este projeto simula o preenchimento de uma matrícula infantil, reunindo informações importantes da criança e do endereço residencial em uma interface moderna, organizada e acolhedora.

A proposta é exercitar conceitos como:

- Estruturação de formulários com **HTML5**
- Estilização com **CSS3**
- Organização de arquivos de estilo
- Uso de **variáveis CSS**
- Estados visuais de foco, campos desabilitados e área de upload

---

## 🧩 Funcionalidades da interface

- Cadastro de informações da criança
- Seleção de data de nascimento
- Escolha de sexo
- Campo para informações médicas
- Upload de certidão de nascimento
- Preenchimento de endereço residencial
- Layout dividido entre formulário e painel institucional

---

## 🚀 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **Google Fonts** (`Poppins`)
